## 目标设定原则 - SMART

目标设定的SMART原则来源于管理大师彼得·德鲁克的《管理的实践》，有五个基本的原则：

- 目标必须是具体的（Specific）
- 目标必须是可以衡量的（Measurable）
- 目标必须是可以达到的（Attainable）
- 目标必须和其他目标具在相关性（Relevant）
- 目标必须具在明确的截止期限（Time-based）
